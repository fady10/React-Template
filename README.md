# react-template
Adding necessary components to start your project with 

<h2> Routing || </h2>

This template comes with routing in the box. Native routing components without any packages. 

<h3>Usages ||</h3> 

```
import Link from 'routing/link'; 

//Showing the route
<Route path='/path-to'>
  <div> Component </div>
</Route>

//Link to the Route 

<Link href='/path-to'>
   My Link 
</Link>
```
